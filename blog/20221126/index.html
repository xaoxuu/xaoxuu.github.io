GitHub Actions 自动化部署

通过 GitHub Actions 可以只提交源码到仓库，后续流程全部由 CI 自动完成，本质上是完成以下几个步骤：

  1. checkout 源码
  2. 配置环境
  3. 生成静态文件
  4. 部署到对应平台（GitHub Pages / Vercel / Server…）

特性

沙盒文件

  • 使用链式语法快速存取沙盒文件

.toJson()

  • .toJson() 快速将字典、数组与 json 进行转换以及非空判断

Blocks

  • 构造方法的 block 支持
  • EventTarget 的 block 支持
  • 给视图添加手势的 block 支持（Runtime）

Cooldown

  • 代码冷却机制（基于 GCD 实现）

Date+

  • NSDate 的链式封装（日期的格式化和操作）
